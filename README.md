# event-countdown-calculator
Problem Description

The aim of this project is to create a Python program that calculates how much time is left for each future event that the user enters. The countdown is shown by the program in days, hours, minutes, and seconds.

Methods Used:

Python built-in modules like datetime and time are used in the program. The event name, future date, and time are entered by the user. The remaining time is calculated and shown repeatedly until the event occurs by comparing the current time in the system with the event time.

Tools Used: 

Python
Time Module  
Datetime module
